    解析: 转化为dp问题
    dp[i]表示以A[i]结尾的最大子串，那么状态转移方程：
    dp[i] = A[i] + dp[i-1] > 0 ? dp[i-1] : 0;

  • maximum product
    解析：转化成dp问题,维护两个数组posArray, negArray
    posArray[i]: 以A[i]结尾的正数最大乘积子串，
    negArray[i]: 以A[i]结尾的负数的最大乘积子串，那么状态转移方程为：
    if(A[i] > 0):
    posArray[i] = Math.max(posArray[i-1]A[i], A[i])
    negArray[i] = negArray[i-1]    A[i]
    else if (A[i] < 0)
    posArray[i] = negArray[i-1]A[i]
    negArray[i] = Math.min(posArray[i-1]    A[i], A[i])
    else (A[i] = 0)
    posArray[i] = 0; negArray[i] = 0;

    Stack st = new Stack();
    Stack output = new Stack();
    st.push(root);

output.push(node.val);
if(node.left != null) st.push(node.left);
if(node.right != null) st.push(node.right);

    }

    res.add(output.pop());

    }
    return res;
